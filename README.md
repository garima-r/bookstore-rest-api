# Bookstore REST API

A Spring Boot backend project built to refresh core backend engineering concepts.

## Features Implemented

- RESTful CRUD APIs
- DTO Layer (decouples API from Entity)
- Bean Validation (Jakarta Validation)
- Global Exception Handling
- Pagination & Sorting (database-level)
- Clean API response design

## Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven
- PostgreSQL

## Why This Project?

Built as part of hands-on learning to strengthen backend fundamentals, focusing on Spring Boot architecture, 
REST API design, validation, and database integration.
